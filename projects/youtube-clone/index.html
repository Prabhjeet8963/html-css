YouTube UI Clone

A responsive recreation of the YouTube interface using HTML and CSS

About This Project

This project is a responsive recreation of the YouTube user interface using HTML and CSS. It demonstrates my ability to implement complex layouts, responsive design, and interactive elements.

Features

  • Responsive header with search functionality
  • Sidebar navigation with hover effects
  • Video grid layout using CSS Grid
  • Custom tooltips and interactive elements
  • Responsive design that adapts to different screen sizes
  • Custom styling for video thumbnails and channel information

Technologies Used

  • HTML5 for structure
  • CSS3 for styling and layout
  • Flexbox for one-dimensional layouts
  • CSS Grid for two-dimensional layouts
  • Media queries for responsive design
  • Google Fonts for typography
